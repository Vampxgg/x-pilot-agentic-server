# Visual Toolbox Summary (Director's Cut)

五大类别 30+ 组件类型，供 ScriptPlanner 在 key_elements 中引用。

## essential-info — 基础信息与陈述
Standard tools for presenting text, lists, and basic information clearly.

| type | displayName | concept |
|------|-------------|---------|
| info-card | Info Card | Highlight a single core concept or title with a brief description and optional icon. Best for introductions or key takeaways. |
| key-value-list | Key-Value List | Display paired attributes, specs, or profiles (e.g., 'Speed: 100km/h'). Ideal for technical specifications. |
| bullet-points | Bullet Points | List steps, features, or multiple items sequentially. Best for summaries or agendas. |
| quote-block | Quote Block | Present authoritative definitions, testimonials, or famous quotes. |
| code-block | Code Block | Show static code snippets or syntax. Essential for technical/programming topics. |
| formula-block | Formula Block | Render mathematical or chemical formulas (LaTeX). Best for academic or scientific explanations. |
| dialogue-block | Dialogue Block | Show a conversation between characters. Good for storytelling or scenarios. |
| kinetic-typography | Kinetic Typography (Advanced) | High-impact, animated text effects (glitch, liquid, reveal). Best for headers, emphatic statements, or energetic transitions. |
| solution-steps | Solution Steps | Walk through a problem-solving process or derivation step-by-step. |
| single-image | Single Image | Display one static image with a caption. Standard for photo slides. |
| image-grid | Image Grid | Display multiple images simultaneously in a grid layout. Good for galleries or collections. |
| annotated-image | Annotated Image | Overlay text labels on specific parts of an image. Ideal for pointing out features on a product photo. |
| before-after-comparison | Before/After Comparison | Compare two states side-by-side (e.g., old vs new, problem vs solution). |

## data-insights — 数据透视与关系
Tools for visualizing data, numbers, charts, and relationships.

| type | displayName | concept |
|------|-------------|---------|
| data-table | Data Table | Structured row/column data presentation. Best when exact values matter. |
| chart | Chart | Statistical visualization (bar, line, pie, etc.). Best for showing trends, comparisons, or distributions. |
| infographic-number | Infographic Number | Dramatically highlight a single key statistic (Big Number). Best for 'Wow' moments. |
| coordinate-system | Coordinate System | Plot mathematical functions or vectors on graph paper. Good for math/physics teaching. |
| vector-field-sim | Vector Field Sim (Advanced) | Visualize invisible forces (wind, gravity, magnetism) as flowing vectors. Best for physics or weather topics. |
| globe-connections-3d | 3D Globe Connections | Show global networks, travel routes, or international relationships on a 3D Earth model. |
| graph-viz | Graph/Network Viz | Show relationships between nodes (knowledge graphs, social networks, topologies). |
| layer-diagram | Layer Diagram | Explain stacked architectures (e.g., OSI model, software layers, geological strata). |
| pyramid-diagram | Pyramid Diagram | Show hierarchical structures (e.g., Maslow's hierarchy, food pyramid). |

## logic-structure — 逻辑、结构与原理
Tools for explaining how things work, structure, logic, and algorithms.

| type | displayName | concept |
|------|-------------|---------|
| schematic-draw | Schematic Blueprint (Enhanced) | Draw technical blueprints or sketches of internal structures. The go-to tool for 'How it works' or engineering breakdowns. |
| model-3d | 3D Model | Showcase a 3D object, potentially with rotation or exploded views. Best for product demos or biological structures. |
| mindmap-growth | Mindmap Growth | Visually branch out ideas from a central root. Good for brainstorming, taxonomy, or evolution of concepts. |
| algo-visualizer | Algorithm Visualizer | Step through algorithmic logic (sorting, searching). Specific to CS education. |
| math-function-visualizer | Math Function Visualizer | Animate dynamic mathematical functions (waves, trajectories). Similar to coordinate-system but focused on animation. |

## simulation-action — 仿真模拟与操作
Tools for simulating real-world physics, actions, or software interfaces.

| type | displayName | concept |
|------|-------------|---------|
| physics-lab-3d | 3D Physics Lab | Simulate physical interactions like gravity, collisions, or pendulums. Best for physics experiments. |
| terminal-simulator | Terminal Simulator | Simulate a hacker/developer command line interface. Best for coding, cybersecurity, or 'tech' vibes. |
| ui-action-sequence | UI Action Sequence | Simulate user interactions (clicks, typing) on a software GUI. Best for tutorials or software demos. |

## visual-metaphor — 视觉隐喻与生成艺术
Abstract and generative tools for high-level concepts and artistic visualization.

| type | displayName | concept |
|------|-------------|---------|
| procedural-canvas | Procedural Canvas (AI Playground) | THE ULTIMATE TOOL. Use for ANY abstract, dynamic, or particle-based visual that doesn't fit elsewhere. Examples: flowing data, biological processes, magical effects, dynamic backgrounds. If you can imagine it as code, this tool can do it. |
| svg-morphing-shape | SVG Morphing Shape | Smoothly transform one shape/concept into another. Perfect for showing 'transformation', 'evolution', or 'change'. |
| particle-system | Particle System | Swarms, flows, or accumulations of small elements. Good for 'crowds', 'dust', 'energy', or 'data gathering'. |
| data-stream-flow | Data Stream Flow | Show high-speed data transmission through pipes/cables. Best for 'internet', 'speed', 'connectivity' themes. |
