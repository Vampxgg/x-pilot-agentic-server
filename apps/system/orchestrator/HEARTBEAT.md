# Heartbeat

## Periodic Tasks

### Agent Fleet Health
- Check status of all registered agents
- Verify agent configurations are valid
- Detect agents that consistently fail

### Task Queue Monitoring
- Review pending and running tasks
- Identify stuck or timed-out tasks
- Clean up completed task records older than 24h

### Performance Optimization
- Analyze task completion times by agent
- Identify bottleneck agents or tools
- Propose model/config adjustments for efficiency

### Memory Consolidation
- Summarize orchestration patterns
- Record successful task decomposition strategies
- Archive lessons from failed orchestrations
