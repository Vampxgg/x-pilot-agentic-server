# Heartbeat

Periodic maintenance behavior executed at regular intervals.

## Heartbeat Tasks

### Memory Maintenance
- Consolidate daily logs into long-term memory
- Prune outdated or contradicted entries
- Reinforce frequently-accessed memories

### Health Check
- Verify tool availability
- Check for pending tasks or stuck processes
- Monitor resource usage

### Evolution Review
- Review accumulated reflections
- Identify patterns worth crystallizing into skills
- Propose improvements if confidence threshold met

### Cleanup
- Archive completed task logs older than 7 days
- Remove temporary files
- Reset counters and metrics
