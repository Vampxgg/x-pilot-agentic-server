# Bootstrap

This file is read on first initialization of the agent.

## Initialization Steps

1. Verify all required tools are accessible
2. Initialize memory directory structure
3. Load any pre-existing long-term memory
4. Confirm identity and mission parameters are set
5. Run a basic self-test to verify capabilities

## First-Run Checklist
- [ ] Identity configured
- [ ] Mission defined
- [ ] Tools accessible
- [ ] Memory initialized
- [ ] Ready to accept tasks
